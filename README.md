# 🍧 Gela Point

Sistema web moderno para gerenciamento de pedidos de açaí, sorvetes e picolés. O projeto foi desenvolvido focado em uma experiência de checkout rápida, fluida e intuitiva para o usuário.

---

## 🚀 Tecnologias Utilizadas

* **HTML5** - Estruturação das páginas e conteúdo.
* **CSS3** - Estilização moderna, responsiva e layout dinâmico.
* **JavaScript (ES6+)** - Lógica do carrinho de compras e validações do checkout.

---

## 🛠️ Como Funciona o Projeto

O projeto conta com uma estrutura organizada e limpa para facilitar futuras manutenções:
* `index.html`: Página principal com a interface dos produtos.
* `estilo.css` & `checkout.css`: Arquivos de estilo responsáveis pela identidade visual.
* `script.js` & `checkout.js`: Responsáveis por processar as interações e a simulação de finalização do pedido.
